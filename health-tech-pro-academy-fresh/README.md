# Health Tech Pro Academy Website

A fresh Vite + React + TypeScript website for **Health Tech Pro**.

## What is included

- Bright academy-style theme inspired by UK online learning websites
- Health Tech Pro branding
- Popular courses section
- Career bundle section
- Digital Care Professional Bundle with all 13 courses
- Working custom bundle builder
- Job Ready Programmes
- Student Adviser enquiry section
- Fully responsive layout
- Vercel-ready build configuration

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Vercel settings

```text
Framework Preset: Vite
Install Command: npm install --no-audit --no-fund
Build Command: npm run build
Output Directory: dist
Node.js Version: 20.x
```

## Important upload note

When uploading to GitHub, upload the full extracted project, including the `src` folder. Your repository must contain:

```text
src
index.html
package.json
tsconfig.json
vite.config.ts
README.md
.npmrc
.gitignore
```

Do not upload the ZIP file itself.
